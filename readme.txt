== Installation ==

You can install the Blocks for Documents, Articles and FAQs plugin from your WordPress Dashboard or manually upload it through FTP.

= OPTION 1: Install the Blocks for Documents, Articles and FAQs plugin from WordPress Dashboard =

1. Navigate to Plugins -> Add New.
2. Search for 'Blocks for Documents, Articles and FAQs' and install the plugin.
3. Activate the plugin in the Plugins menu.

= OPTION 2: Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: blocks-for-documents-articles-and-faqs.zip.
2. Unzip the file blocks-for-documents-articles-and-faqs.zip.
3. Upload the 'blocks-for-documents-articles-and-faqs' folder to your '/wp-content/plugins' directory (do not rename the folder).
